#목표
자본 추적(?) 및 타겟 마케팅전략(숨은 패턴) 도출

## 기준
- 동네를 잘 모르는 사람도 전략을 이해할 수 있어야 함.
- 그래프가 이해하기 쉽고 타당하며 가독성, 심미성, 전문성을 챙겼는가?
- 신뢰성이 있는 자료인가?(R 통계검증)
- 코드의 비효율성이 있었는가 ? 
- 인사이트를 도출하고 전략을 수립했는가 ? 
- 사용자의 Python 및 데이터 분석 스킬이 늘었는가 ? 

#프로젝트 개요

 - 월 -> 주 -> 일 -> 월(시간보고) 로 진행


#월 단위 ( 한마디로 수박겉핥기 )
- 이중 축 그래프로 전체 매출확인
- 기간(초,중,말) 별 매출총액 비교(6월 8월은 어떠할까? )
- 요일 별(월~일) 매출총액 비교(6월 8월은 어떠할까?, 왜 낮지 ? ) 
- 요일 + 시간대 히트맵
- 연령 + 시간대 히트맵

Drill-Down
 - 주제 : 음식
Topic1). 이중축 그래프로 전체적인 틀을 확인(뭘 도출할건데?, 무슨 가설을 수립할건데?)
Topic2). 남/녀 남자의 결제주도 가설

#주 단위
- 월 그룹화(월초, 중순, 월말) 매출총액/매출건수 차이 

#일 단위 ( 상세 내역 ) 

#월(다른 월의 경향은 어떠한가 ? 전체적으로 어떤 흐름인가?)
- 만약 음식에서 왜 내려가는 패턴인지(월급인지 더워서인지) 밝히지 못했다면 다른 월과 비교하면서 검증해야함.
